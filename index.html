About the Holding

Opaleon Holdings is a Delaware C-Corporation that owns the IP and provides shared security, finance, legal, and data infrastructure for four independent software companies. We optimize for optionality - build together yet allow clean exits when timing is right.

Focus

Mortgage-first software that removes bottlenecks and speeds up origination. We build where brokers feel the most friction.

Security

PCI compliance, SOC 2 practices, end-to-end encryption, and disaster recovery patterns that each company inherits.

Principles

Speed, clarity, and service. Truth over hype. Practical integrations before flashy features. Broker success is our north star.

Structure

Delaware incorporation with remote-first operations. Tax-optimized structure with clear paths for each subsidiary's growth.
